Perguntas Frequentes

Quais são os requisitos de preparação?

Recomendamos usar calçados confortáveis para caminhada, vestuário adequado às condições climáticas e trazer água e lanches.

Há alguma restrição de idade?

A atividade é adequada para participantes de todas as idades, no entanto, menores de idade devem estar acompanhados por um adulto.

Posso participar sozinho?

Sim, nossas trilhas são projetadas para receber tanto indivíduos quanto grupos.

É necessário ter equipamento próprio para participar?

Não é necessário ter seu próprio equipamento. Fornecemos todos os equipamentos de segurança necessários e quaisquer outros equipamentos específicos para a trilha.
